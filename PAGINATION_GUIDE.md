# WangScape 分页功能指南

## 功能概述

WangScape博客系统现已集成完整的分页功能，支持优雅的文章导航和丰富的交互体验。

## 🚀 核心功能

### 1. 自动分页
- 每页显示 **3篇文章**（可在 `config/_default/config.toml` 中调整）
- 自动生成分页导航：`/page/1/`, `/page/2/`, 等
- 智能显示省略号，优化导航体验

### 2. 视觉设计
- 现代玻璃态设计，支持明暗主题
- 平滑的贝塞尔曲线动画过渡
- 响应式布局，移动端优化
- 当前页面高亮显示
- 禁用状态清晰标识

### 3. 交互增强
- **键盘导航**：使用 `←` `→` 键快速切换页面
- **平滑过渡**：页面切换时的淡入淡出效果
- **加载状态**：点击分页按钮时的加载动画
- **悬停效果**：微交互动画提升体验
- **滚动记忆**：返回时保持滚动位置

## 📱 使用方法

### 基本导航
1. **鼠标操作**：点击分页按钮进行导航
2. **键盘操作**：
   - `←` 上一页
   - `→` 下一页
   - `ESC` 退出全屏等交互

### 分页组件说明
```
[‹ Prev] [1] [2] [3] ... [10] [Next ›]  Page 3 of 10
```
- **箭头按钮**：上一页/下一页
- **数字按钮**：直接跳转到指定页面
- **省略号**：表示中间有更多页面
- **页面信息**：显示当前页码和总页数

## ⚙️ 配置选项

### 调整每页文章数量
编辑 `config/_default/config.toml`：
```toml
[pagination]
pagerSize = 5  # 改为你想要的数量
```

### 自定义分页路径
```toml
[pagination]
pagerSize = 3
disableAliases = false
path = 'page'  # 分页URL路径
```

## 🎨 样式自定义

分页样式位于 `assets/scss/custom.scss`：

### 主要CSS类
- `.pagination`：分页容器
- `.page-link`：分页按钮
- `.current`：当前页面
- `.disabled`：禁用状态
- `.pagination-info`：页面信息显示

### 暗色模式
自动支持暗色主题，样式会根据 `data-scheme` 属性切换。

## 📄 模板结构

### 自定义分页模板
位置：`layouts/partials/pagination.html`

支持的变量：
- `$pag.PageNumber`：当前页码
- `$pag.TotalPages`：总页数
- `$pag.Prev`：上一页链接
- `$pag.Next`：下一页链接
- `$pag.First`：第一页链接
- `$pag.Last`：最后一页链接

## 🔧 JavaScript增强

### 自动加载的增强功能
- **平滑过渡**：页面切换动画
- **键盘导航**：方向键支持
- **加载状态**：点击反馈
- **滚动管理**：位置记忆和自动滚动

### 禁用JavaScript
即使JavaScript被禁用，分页功能仍然正常工作，只是缺少一些交互增强。

## 🌐 多语言支持

分页支持中英文界面：
- 中文：上一页/下一页，第X页，共Y页
- 英文：Prev/Next，Page X of Y

## 📱 响应式设计

### 桌面端（>768px）
- 完整分页导航
- 文字按钮 + 箭头
- 页面信息显示
- 键盘导航提示

### 移动端（≤768px）
- 紧凑布局
- 仅显示箭头图标
- 隐藏文字标签
- 隐藏键盘提示

## 🚀 部署注意事项

1. **静态文件**：确保 `js/pagination-enhanced.js` 被正确部署
2. **权限**：检查文件访问权限
3. **CDN**：如使用CDN，确保JavaScript文件被缓存
4. **测试**：在不同浏览器和设备上测试功能

## 🔍 故障排除

### 分页不显示
1. 检查文章数量是否超过 `pagerSize`
2. 验证 `config.toml` 中的分页配置
3. 确认 `layouts/partials/pagination.html` 存在

### JavaScript功能失效
1. 检查浏览器控制台错误
2. 验证 `js/pagination-enhanced.js` 文件路径
3. 确认没有JavaScript冲突

### 样式问题
1. 清除浏览器缓存
2. 检查CSS编译是否成功
3. 验证主题CSS优先级
